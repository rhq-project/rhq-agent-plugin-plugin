Deploying to a local RHQ server

You will most probably install an RHQ Server and an Agent on your development box in order to test your Agent plugin. The deploy goal will copy your freshly built plugin to your local server plugins directory.

Plugin Execution Sample

    ...
    <configuration>
        <deployDirectory>/path/to/dev/container/plugins/dir</deployDirectory>
    </configuration>
    <execution>
        <id>deploy-to-dev-container</id>
        <goals>
            <goal>deploy</goal>
        </goals>
        <phase>package</phase>
    </execution>
    ...
