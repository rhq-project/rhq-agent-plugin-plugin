How to use the plugin?

From v0.5 and on

The plugin is published on Maven Central and on JBoss Releases repositories. You no longer need to configure your Maven settings file.

Before v0.5

Before v0.5, the plugin was published on Bintray.

You must configure your Maven settings file to enable the Bintray repository:

<settings>
 ...
  <profiles>
    ...
    <profile>
      <id>tsegismont-rhq-maven-plugins</id>
      <pluginRepositories>
        <pluginRepository>
          <id>tsegismont-rhq-maven-plugins</id>
          <url>https://dl.bintray.com/content/tsegismont/rhq-maven-plugins</url>
        </pluginRepository>
      </pluginRepositories>
    </profile>
    ...
  </profiles>
  ...
  <activeProfiles>
    ...
    <activeProfile>tsegismont-rhq-maven-plugins</activeProfile>
    ...
  </activeProfiles>
 ...
</settings>
