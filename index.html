RHQ Agent Plugin Plugin

Welcome to the RHQ Agent Plugin Plugin documentation.

Plugin Plugin?

Yes. Plugin Plugin: a Maven Plugin to help you build RHQ Agent Plugins.

Great. But I know nothing about RHQ Agent Plugings...

Then you'll want to read the Getting Started with RHQ Agent Plugins wiki page first. You can also checkout the How to write a plugin for JBoss ON and RHQ Guide (PDF) - or the more up to date version How to write a plugin.

Available features

  • Packaging
  • Validation
  • Deployment to a local RHQ Server (dev-container)
  • Upload to to a remote RHQ Server
  • CLI script or command execution
  • Standalone Plugin Container setup for integration tests
  • Support for dependent plugins (with respect to validation and test Plugin Container setup)

Planned

  • Agent Plugin project generation
  • Plugin Documentation generation
